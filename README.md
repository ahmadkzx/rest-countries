## 🚀 Local Version

Requirements:
 - node
 - yarn
 - @vue/cli
 - docker

### Run Manually
```bash
# install dependencies
yarn

# serve on localhost:8080
yarn serve
```



### Run With Docker
```bash
# build image
yarn docker:build

# run image on localhost:8080
yarn docker:run 
```
